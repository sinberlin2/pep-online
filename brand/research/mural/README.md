# Mural exports for PEP branding

Drop files here after exporting from your board:

| File | How |
|------|-----|
| `board.png` | Mural → Export frame/board as PNG |
| `use-cases.png`, `brand-map.png` | Optional section exports |
| `notes.md` | Pasted sticky-note text (optional) |
| `link.txt` | Board share URL (optional, for your reference) |

See **`../IMPORT.md`** for full instructions.

**Do not commit** huge PDFs; PNG at reasonable resolution is enough.
